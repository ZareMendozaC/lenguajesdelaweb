Algunos Lenguajes de la Web

HTML + CSS + JavaScript

  • HTML5

    • HTML5 (HyperText Markup Language, versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML.
    Usa la extensión .html

  • CSS

    • Hoja de estilos en cascada o CSS (Siglas en Inglés de cascading style sheets) es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.
    Usa la extensión .css

  • JavaScript

    • JavaScript (abreviado comumente como "JS") es un lenguaje de programación interpretado utilizado principalmente en el lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas.
    Usa la extensión .js
